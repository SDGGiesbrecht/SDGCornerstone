SDGCornerstone SearchableCollection advance(_:over:)

advance(_:over:)

Advances the index over the pattern.

Declaration

func advance(_ index: inout Index, over pattern: CompositePattern<Element>) -> Bool

Overview

  • Parameters:

    • index: The index to advance.

    • pattern: The pattern to advance over.

  • Returns

    true if the index was advanced over a match, false if there was no match.
