SDGCornerstone SearchableCollection truncated(before:)
Instance Method

truncated(before:)

Returns a collection formed by truncating self at the start of the first match for the specified pattern.

Declaration

func truncated(before pattern: CompositePattern<Element>) -> Self where Self : RangeReplaceableCollection

Discussion

If the pattern does not occur, the collection will remain unchanged.

  • Parameters:

    • pattern: The pattern to search for.
