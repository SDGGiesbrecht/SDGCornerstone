SDGCornerstone SearchableCollection drop(through:)
Instance Method

drop(through:)

Drops elements from the beginning of the collection to the end of the first match for the specified pattern.

Declaration

mutating func drop(through pattern: CompositePattern<Element>) where Self : RangeReplaceableCollection

Overview

If the pattern does not occur, the collection will empty itself.

  • Parameters:

    • pattern: The pattern to search for.
