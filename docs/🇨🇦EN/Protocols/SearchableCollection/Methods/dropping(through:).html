SDGCornerstone SearchableCollection dropping(through:)
Instance Method

dropping(through:)

Returns a collection formed by dropping the elements from the beginning of the collection to the end of the first match for the pattern.

Declaration

func dropping(through pattern: CompositePattern<Element>) -> Self where Self : RangeReplaceableCollection

Discussion

If the pattern does not occur, the collection will empty itself.

  • Parameters:

    • pattern: The pattern to search for.
