Instance Method

dividedAccordingToEuclid(by:)

Returns the integral quotient of self divided by divisor.

Declaration

func dividedAccordingToEuclid(by divisor: Self) -> Self

Overview

  • Note

    This is a true mathematical quotient. i.e. (−5) ÷ 3 = −2 remainder 1, not −1 remainder −2

  • Parameters:

    • divisor: The divisor.
