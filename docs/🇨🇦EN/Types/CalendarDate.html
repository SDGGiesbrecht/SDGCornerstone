Structure

CalendarDate

A date on a particular calendar.

Declaration

struct CalendarDate

Overview

The CalendarDate structure will remain accurate to its initial definition even if calendar or time zone rules change in the future, such as a change in the Daylight Savings start or end times. (This is in contrast to Date, which simply defines itself by a number of seconds since an epoch. If any rules were changed in the future, converting it back to a calendar‚Äêbased representation would result in a different date and time.)

To get the current time, use a static function such as hebrewNow() or gregorianNow(). Each returns a date representing the current time, but is defined according to a different calendar.
