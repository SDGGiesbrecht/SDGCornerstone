Initializer

init(unsafeRawValue:)

Creates an instance with an unchecked raw value.

Declaration

init(unsafeRawValue: RawValue)

Overview

  • Note

    Do not call this initializer directly. Call init(_:) instead, because it validates the raw value before passing it to this initializer.
