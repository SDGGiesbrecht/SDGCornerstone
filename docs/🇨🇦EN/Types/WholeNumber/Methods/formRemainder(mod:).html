SDGCornerstone WholeNumber formRemainder(mod:)
Instance Method

formRemainder(mod:)

Sets self to the Euclidean remainder of self ÷ divisor.

Declaration

mutating func formRemainder(mod divisor: WholeNumber)

Overview

  • Parameters:

    • divisor: The divisor.

  • Note

    This is a true mathematical modulo operation. i.e. (−5) mod 3 = 1, not −2
