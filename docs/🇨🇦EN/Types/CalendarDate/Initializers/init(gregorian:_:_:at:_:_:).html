SDGCornerstone CalendarDate init(gregorian:_:_:at:_:_:)
Initializer

init(gregorian:_:_:at:_:_:)

Creates a date using the Hebrew calendar.

Declaration

init(gregorian month: GregorianMonth, _ day: GregorianDay, _ year: GregorianYear, at hour: GregorianHour = default, _ minute: GregorianMinute = default, _ second: GregorianSecond = default)

Overview

This initializer has two written forms whose effects are the same:

  • init(gregorianYear:month:day:hour:minute:second:) tends to be more legible when used with variables.

  • init(gregorian:_:_:at:_:_:) tends to be more legible when used with literals.
