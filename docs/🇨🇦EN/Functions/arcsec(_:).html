SDGCornerstone arcsec(_:)
Function

arcsec(_:)

Returns the arcsecant of a value.

Declaration

func arcsec(_ secant: N) -> RealArithmetic

Discussion

The returned angle will be between 0° and 180° inclusive, but never 90°.

  • Precondition

    −1≥ secant ∨ secant ≤ 1

  • Parameters:

    • secant: The secant.
