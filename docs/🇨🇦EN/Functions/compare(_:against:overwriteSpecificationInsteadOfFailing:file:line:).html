Function

compare(_:against:overwriteSpecificationInsteadOfFailing:file:line:)

Compares a string against a test specification.

Declaration

func compare(_ string: String, against specification: URL, overwriteSpecificationInsteadOfFailing: Bool, file: StaticString = default, line: UInt = default)

Overview

If the string does not match the specification, the test will fail.

If the specification does not exist yet, it will be created according to the string. It is recommeded to check specifications into source control to monitor changes.

To update a specification instead of testing against it, change overwriteSpecificationInsteadOfFailing to true and re‚Äêrun the test suite. The specification will be rewritten to match the provided string. Do not forget to change it back afterward, or the test will cease to validate anything.

  • Parameters:

    • string: The string to test.

    • specification: The location of the specification to compare against (or write to).

    • overwriteSpecificationInsteadOfFailing: Set to false for normal behaviour. Set to true temporarily to update a specification.
