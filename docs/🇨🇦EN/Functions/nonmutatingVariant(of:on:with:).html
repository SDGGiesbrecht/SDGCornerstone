nonmutatingVariant(of:on:with:)

Implements a nonmutating function based on its mutating counterpart.

Declaration

func nonmutatingVariant(of mutation: ->, on self: T, with argument: A) -> T

Discussion

extension Array where Element : Comparable { // MARK: - where Element : Comparable func sorted() -> Array { return nonmutatingVariant(of: Array.sort, on: self) } func appending(_ appendix: Array) -> Array { return nonmutatingVariant(of: Array.append, on: self, with: appendix) } static func + (a: Array, b: Array) -> Array { return nonmutatingVariant(of: +=, on: a, with: b) } }

    • mutation: The mutating counterpart.
