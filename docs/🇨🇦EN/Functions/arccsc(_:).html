Function

arccsc(_:)

Returns the arccosecant of a value.

Declaration

func arccsc(_ cosecant: N) -> RealArithmetic

Discussion

The returned angle will be between −90° and 90° inclusive, but never 0°.

  • Precondition

    −1 ≥ cosecant ∨ cosecant ≤ 1

  • Parameters:

    • cosecant: The cosecant.
