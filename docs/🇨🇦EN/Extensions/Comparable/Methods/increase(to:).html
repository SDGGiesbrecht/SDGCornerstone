Instance Method

increase(to:)

Increases the value of self so that falls at or above minimum.

Declaration

mutating func increase(to minimum: Self)

Discussion

This is accomplished by changing self to match the value of minimum if and only if self does not already satisfy self ≥ minimum.

For example:

func rollDie() -> Int { return Int(randomInRange: 1 ... 6) } let numberOfRolls = 5 var highestRoll = 1 for _ in 1 ... numberOfRolls { highestRoll.increase(to: rollDie()) } print("After rolling the die \(numberOfRolls.inDigits()) time(s), the highest roll was \(highestRoll.inDigits()).") // Prints, for example, “After rolling the die 5 time(s), the highest roll was 4.” // In each iteration of the for loop, a new number is rolled, and if it is greater than highestRoll’s existing value, increase(to:) changes highestRoll to reflect the new high.

  • Parameters:

    • minimum: The desired minimum for the value.
