Instance Method

runForDriver(_:)

Runs the run loop for the lifetime of the driver provided to holdDriver.

Declaration

func runForDriver(_ holdDriver: ->)

Discussion

For example:

var driver: RunLoop.Driver? DispatchQueue.global(qos: .userInitiated).async { RunLoop.current.runForDriver { driver = $0 } } // The background run loop is now running. driver = nil // The background run loop has now stopped.
  • Warning

    Giving the run loop (or any of its timers, ports, etc.) a strong reference to the driver will create a retain cycle.

  • Parameters:

    • holdDriver: A closer that takes ownershipe of the driver by creating a strong reference to it somewhere with the desired lifetime.

    • driver: The driver that runs the loop. As soon as ARC deallocates this driver, the run loop will stop.
