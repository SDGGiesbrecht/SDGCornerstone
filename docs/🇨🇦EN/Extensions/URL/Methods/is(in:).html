SDGCornerstone URL is(in:)
Instance Method

is(in:)

Returns true if the URL is in the location described by the specified URL.

Declaration

func is(in other: URL) -> Bool

Overview

The URL is considered to be inside if it:- points to a file or subfolder of the other URL, or- points to the same file or folder as the other URL.

  • Parameters:

    • other: Another URL.
