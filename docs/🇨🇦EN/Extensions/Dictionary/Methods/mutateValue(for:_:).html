mutateValue(for:_:)

Mutates the value for the specified key.

Declaration

mutating func mutateValue(for key: Key, _ mutate: throws)

Discussion

func rollDie() -> Int { return Int(randomInRange: 1 ... 6) } var frequencies = [Int: Int]() for _ in 1 ... 100 { frequencies.mutateValue(for: rollDie()) { ($0 ?? 0) + 1 } } print(frequencies.keys.sorted().map({ "\($0.inDigits()): \(frequencies[$0]!.inDigits())" }).joined(separator: "\n")) // Prints, for example: // // 1: 21 // 2: 8 // 3: 29 // 4: 12 // 5: 20 // 6: 10 // In this example, the die is rolled 100 times, and each time the tally for the outcome is incremented. After the for loop, the dictionary contains the frequencies (values) for each outcome (keys).

  • Parameters:

    • key: The key whose value should change.

    • mutate: A mutating closure.

    • previous: The previous value or nil if the dictionary does not contain the specified key.
