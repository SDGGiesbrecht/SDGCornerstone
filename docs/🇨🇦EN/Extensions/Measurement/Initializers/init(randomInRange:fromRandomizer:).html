SDGCornerstone Measurement init(randomInRange:fromRandomizer:)
Initializer

init(randomInRange:fromRandomizer:)

Creates a random value within a particular range using the specified randomizer.

Declaration

init(randomInRange range: Range<Self>, fromRandomizer randomizer: Randomizer) where Scalar : RandomizableNumber

Overview

  • Precondition

    range is not empty.

  • Parameters:

    • range: The allowed range for the random value.

    • randomizer: The randomizer to use to generate the random value.
